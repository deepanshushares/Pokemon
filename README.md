

# Pokemon App

This is a simple application that show us different **Pokemons** randomly when the CTA button is tapped.


# Concepts used

- UIImageView
- UIButton
- UILabel
- Arrays
- Life Cycle Methods
- Optionals, Optional Binding
- Running your first application on an iPhone
